### Layout Update: Walkthrough Panel Position

Update the Explore Level 2 layout as follows:

1. **Reposition the walkthrough task window**

   * Dock the walkthrough task window to the **left side** of the interactive display.
   * The task window should remain visible while the student interacts with the display.

2. **Preserve all existing functionality**

   * Do not change the behavior of the walkthrough system, the interactive display, or any validation logic.
   * Only modify the layout.

3. **Optimize the layout for a single-screen view**

   * Arrange the interface so that the following elements are simultaneously visible without scrolling or switching views:

     * Walkthrough task window
     * Interactive display
     * Equations panel
     * Component Breakdown panel

4. **Responsive sizing**

   * Resize the walkthrough panel as needed to maximize the available workspace while keeping all task content readable.
   * The interactive display should retain as much usable area as possible while allowing all four interface components to remain visible on a standard desktop display.
