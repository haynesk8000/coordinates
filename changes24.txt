Update the **Fun Zone** tab in **Coordinate Systems** so that difficulty scales automatically based on student performance.

### Difficulty Rules

* Every activity starts at **0% difficulty**.
* Track correct responses within the current challenge.
* After the student gives **3 correct responses**, increase the difficulty by **20%**.
* Continue increasing difficulty by **20%** after every additional set of **3 correct responses**.
* Difficulty must not exceed **100%**.

### Difficulty Progression

* 0 correct responses: **0%**
* 3 correct responses: **20%**
* 6 correct responses: **40%**
* 9 correct responses: **60%**
* 12 correct responses: **80%**
* 15 or more correct responses: **100%**

### Implementation Notes

* Apply this behavior to all Fun Zone activities.
* Each difficulty increase should make the challenge meaningfully harder.
* Adjust challenge difficulty using one or more of the following, depending on the activity:

  * problem complexity
  * coordinate range
  * number of objects or targets
  * precision required
  * time limits
  * number of steps or transformations
  * mixed concept combinations

### Goal

The difficulty system should create a smooth progression from easy to hard while keeping the activities fun, playable, and engaging at every level.
