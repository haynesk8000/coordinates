## Task: Create Guided Explore Activities

Based on the application's purpose, features, and learning objectives, create a set of guided activities that help students explore how changing the coordinate system affects the mathematical description of projectile motion.

### Deliverables

Create:

* **10 Translation tasks** (moving the origin)
* **10 Rotation tasks** (rotating the coordinate axes)

Save all tasks in a Markdown file named **`explorer_tasks.md`**.

### Task Format

Each task should include the following sections:

#### Title

A short, descriptive title.

#### Description

Provide 3 to 5 clear, step-by-step instructions telling the student exactly what to do in the Explore tab.

#### Observation

Describe what the student should notice after completing the steps. Focus on changes in the displayed coordinates, equations, vectors, or other quantities.

#### Explanation

Briefly explain **why** the observed changes occur. Emphasize that changing the coordinate system changes the mathematical description of the motion, **not the physical motion itself**.

### Guidelines

* Begin with simple tasks and gradually increase complexity.
* Cover translations in multiple directions (left, right, up, down, and combinations).
* Cover a variety of rotation angles, including positive and negative rotations.
* Encourage students to compare the equations before and after moving or rotating the axes.
* Highlight coordinate invariance where appropriate, distinguishing between quantities that change with the coordinate system and those that do not.
* Keep descriptions concise and focused on exploration rather than calculation.
* Explanations should be accurate but brief (1 to 2 short paragraphs).
* Avoid requiring numerical calculations unless they directly support the learning objective.

### Example

**Title:** Translation to the Upper Right

**Description**

1. Place the origin at the launch point and observe the initial coordinates.
2. Move the origin upward and to the right.
3. Observe how the values of (x_0) and (y_0) change.
4. Explain why the initial position is now represented by negative values.

**Observation**

After moving the origin, the launch point is located to the left and below the new origin. The displayed initial coordinates become negative, even though the projectile follows the same path.

**Explanation**

Moving the origin changes only the coordinate system. The projectile has not moved, but its position is now measured relative to a different reference point. The negative signs indicate that the launch point lies left of and below the new origin, not that the projectile's motion has changed.
